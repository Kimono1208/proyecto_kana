Problematica

La empresa tiene un problema con el control de sus productos, ventas, facturación y proveedores.

  • El control de inventarios se realiza mediante hojas de cálculo, las cuales no se actualizan con frecuencia y presentan inconsistencias.
  • No se registra correctamente la salida del producto, y hay fugas y pérdidas de material.
  • El control de proveedores también se realiza mediante hojas de cálculo, las cuales no cuentan con información de los productos que ofrece cada proveedor.
  • Las ventas y facturación se realizan en caja y a mano, lo que genera demoras e inconvenientes para los clientes.

Solucion implementada

Implementamos programas y procesos que eliminan o reducen el error humano como un factor.

  • Inventarios:

    Se desarrollo un programa que pide al usuario registrar compras al momento de la llegada de la mercancia, lo que mantiene los inventarios actualizados.

  • Salida de productos:

    Se desarrollo una hoja de calculo automatizada en la que el balance de entradas y salidas se calcula automaticamente para retornar al usuario un inventario exacto y confiable en todo momento.

  • Control de proveedores:

    Se desarrollo una interfaz en Java para que el usuario pueda registrar de forma rapida y efectiva a los provedores que arriban a la tienda, esta interfaz interactua con una hoja de excel automatizada que permite almacenar la informacion y acceder a ella en todo momento.

  • Ventas y facturacion:

    El equipo de KANA diseño e implemento una hoja de calculo con macros que reduce el tiempo de facturacion accediendo a la base de datos de clientes registrados que se actualiza diariamente mediante el programa de Java para autocompletar los datos de facturacion.
